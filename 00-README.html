The Functional Parsing Library

In this library I, Peter Ljunglöf, have collected Haskell files for the parsers and parsing algorithms described in my Licenciate Thesis. Observe that you will need the following extensions of Haskell 98:

  • Multi-parameter type classes with functional dependencies
  • Existential quantification in data types
The files here are tested in both Hugs and GHC (with the appropriate flags for turning on Haskell extensions).

Any questions or comments on the code or the thesis are of course much appreciated.

Combinator Parsers  (ch. 3 and 4)

The parsers described in these chapters can be found in the parsers directory. Apart from the files in that directory you will need the OrdMap module from this directory.

Parsing Algorithms  (ch. 5, 6 and 7)

The parsers described in these chapters can be found in the algorithms directory. Apart from the files in that directory you will need the OrdSet and OrdMap modules from this directory.

The Grammar Collecting Parser  (ch. 8)

The grammar collecting parser makes use of files from various places, apart from the module of your choice from the collecting-parser directory. First you need the OrdSet and OrdMap modules from this directory; second you need the Parser module from the parsers directory; and third you need the appropriate modules from the algorithms directory, to be able to run the chosen parsing algorithm.
