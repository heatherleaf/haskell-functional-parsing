The Functional Parsing Library

The Grammar Collecting Parser (ch. 8)

To use the grammar collecting parser described in this chapter, you first need the following files:
  • OrdSet.hs   (from the parent directory)
  • OrdMap.hs   (from the parent directory)
  • Parser.hs   (from the parsers directory)
  • Grammar.hs   (from the algorithms directory)
  • Ref.hs
Then you choose which variant of the collecting parser you want to use:
  1. CollectingParserOriginal.hs   (the code from section 8.2; only works with chart parsing)

  2. CollectingParserOptimized.hs   (optimized version for efficiency and generality; works with chart parsing and LR parsing)

  3. CollectingParserToChart.hs   (uses the parsing algorithms that parses to a chart of passive edges; works with chart parsing, LR parsing and CYK parsing)
Finally you need an implementation of a parsing algorithm, from the algorithms  directory.

Observe that since the Ref module uses functions from the IOExts module, you have to include the lang package when compiling with GHC. I.e. you need to add the argument -package lang to the GHC compiler.
